单例（ Single Instance ）

1. 定义

单例模式（Singleton Pattern）涉及到一个单一的类，该类负责创建自己的对象，同时确保只有单个对象被创建。这个类提供了一种访问其唯一的对象的方式，可以直接访问，不需要实例化该类的对象。

2. Object-C 中的单例模式

@implementation TNSQLiteManager

+ (TNSQLiteManager *)sharedManager {
  	static TNSQLiteManager *sharedManager = nil;
    static dispatch_once_t onceToken;
    dispatch_once(&onceToken, ^{
        sharedManager = [[super allocWithZone:NULL] init] ;
    });
    return sharedManager;
}

+ (id)allocWithZone:(struct _NSZone *)zone {
    return [TNSQLiteManager sharedManager] ;
}

- (id)copyWithZone:(struct _NSZone *)zone {
    return self ;
}

@end
  • [[super allocWithZone:NULL] init]这里一定要用super，如果用self会调用allocWithZone产生死循环；
  • 要重写allocWithZone，防止开发者用alloc来初始化这个对象；
  • 要重写copyWithZone，防止开发者用copy来拷贝这个对象；

3. Swift中的单例模式

class TNSQLiteManager {
    
    /// 单例，全局数据库工具访问点
    static var shared = TNSQLiteManager()
    ...
}
